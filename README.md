### Local Dev Setup

1. Open Ganache Blockchain. 

```
ganache cli
```

2. Install dependencies

```
npm install
```

3. Deploy smart contracts

```
truffle migrate
```

4. Start local react app

```
npm start
```

*Note: This application has not been deployed on Ethereum MainNet.*
